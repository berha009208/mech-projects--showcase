Compact Air Compressor System Design| By Ariyam Berhe

Project Overview

This project involved the design and testing of a compact, low-power air compressor system for general applications. While a bicycle tyre was used as an analogy, the design focused on broader use cases such as portability, reliability, and efficient performance.

Design Features

  • CAD Modeling: SolidWorks model with piston-cylinder + crank-slider mechanism.
  • Manufacturing: 3D printed in Onyx — strength + airflow integration.
  • Sealing: Dynamic & static seals to minimize leakage.
  • Thermodynamics: Work analysis using adiabatic/isothermal models.
  • Testing: Achieved 0.82 bar @ 15V with suggestions for improvements.

Testing & Improvements

  • Fixed issues with shaft misalignment and baseplate deformation.

  • Future aim: consistent 1 bar pressure with minimal energy loss.
